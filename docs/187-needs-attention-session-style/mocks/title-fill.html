White text on the current token (amber-600) sits around 2.6:1 contrast — under the 4.5:1 AA bar for small text — so it reads a bit washed. amber-700 (~3.4:1) and amber-800 (~4.8:1) keep the same hue but make the white title legible; #1 uses 800. The highlight hugs the title and truncates with it, and because only the title is filled it coexists cleanly with the gray selected row (row 4).
