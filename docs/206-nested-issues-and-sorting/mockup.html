What changed from the first prototype
  • Recursive nesting: parents nest to any depth (SHI-90 → SHI-93 → SHI-94 → SHI-95). Each level adds a tree spine; collapsing any node hides its whole subtree.
  • Modal editor: the sliders icon opens the two-level sort + group-by editor. The icon shows an accent dot when the order differs from the default, and a one-line summary appears under the toolbar so the active order is visible without opening the modal.
  • Independent Group-by: with the modal's extra room, grouping is its own field (None / Priority / Status / Assignee), not tied to the primary sort key.
  • Mobile = indent only: the card layout drops disclosure triangles and tree lines; a child card is indented with a left rule. (Confirmed direction — no collapse/expand at this width.)
  • GitHub stays flat: no parent data, so the GitHub tab renders today's flat list — nothing to build there.
